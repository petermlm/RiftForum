BBCode

These are the various bbcode tags which can be used to style Messages and Signatures. Most are available directly from the library that implements them. Others were developed as custom tags. Those are marked with a start (*).
Rule Description 
[b]text[/b]
 Makes text bold 
[url]link[/url]
 Makes url a link 
[url=link]text[/url]
 Makes url a link with a different text 
[img]link[/img]
 Places an image 
[img=link]alt[/img]
 Places an image with alt text 
[center]text[/center]
 Center content enclosed in tags 
[color=red]text[/color]
 Colors text with given color 
[size=2]text[/size]
 Sets size of text 
[quote]text[/quote]
 Displays text quoted 
[quote=Somebody]text[/quote]
 Displays text quoted and a name of quoted 
[quote name=Somebody]text[/quote]
 Same as previous 
[code][b]anything[/b][/code]
 Any bbcode inside the code tags will shown literally 
[list][*]anything[/list]
 Displays a list in which each entry is given by [*]
